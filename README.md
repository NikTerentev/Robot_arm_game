# [![Typing SVG](https://readme-typing-svg.herokuapp.com/?lines=Robot+arm+game)](https://git.io/typing-svg)

## [![Typing SVG](https://readme-typing-svg.herokuapp.com/?lines=Цель+проекта)](https://git.io/typing-svg)
